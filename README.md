# we-share
Java EE course project  

## Install Mavon Editor  
```$xslt
npm install mavon-editor --save  
```
see  
https://www.jianshu.com/p/04376d0c9ff1  
https://www.jianshu.com/p/474197c1f7cc  

## Install iView  
```$xslt
npm install view-design --save  
```
see https://www.iviewui.com/
